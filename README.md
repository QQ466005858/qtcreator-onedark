Onedark theme for Qt Creator
=================================

Installation
------------
Installation on Linux/Mac:

    cp onedark.xml ~/.config/QtProject/qtcreator/styles/
Installation on Windows 7:

    xcopy onedark.xml %APPDATA%\QtProject\qtcreator\styles
